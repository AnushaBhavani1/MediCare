 MediCare

MediCare is a healthcare management web application built to simplify the interaction between patients and doctors. The main goal of this project is to make healthcare services more organized and accessible through an online platform.

This system allows patients to register, book appointments, and view healthcare services, while doctors can manage patient details and appointments easily.

Features

- User registration and login
- Doctor and patient dashboards
- Appointment booking system
- Healthcare service management
- Secure authentication
- Responsive user interface
